1. Introdução à Plataforma .NET e C#

Bem-vindo(a) à sua porta de entrada para o mundo do desenvolvimento de software com a plataforma **.NET** e a linguagem **C# (C Sharp)**! Esta apostila foi cuidadosamente elaborada para ser seu guia completo, prático e atualizado, ideal para quem está começando e deseja construir uma base sólida para se tornar um desenvolvedor júnior de sucesso.

O universo da programação é vasto e dinâmico, e escolher uma tecnologia robusta e amplamente utilizada é um passo fundamental. A plataforma .NET, criada e mantida pela Microsoft, junto com C#, oferece um ecossistema poderoso, versátil e com vasta documentação, sendo uma das escolhas mais populares para o desenvolvimento de aplicações corporativas, web, mobile, jogos e muito mais.

1.1. O Que é .NET?

A plataforma **.NET** é um framework (estrutura) de desenvolvimento de software **gratuito e de código aberto**, utilizado para construir uma ampla variedade de aplicações. Pense nele como um grande conjunto de ferramentas, bibliotecas e um ambiente de execução que permite aos desenvolvedores criar diversos tipos de sistemas de forma eficiente e segura.

Desde aplicações web de alta performance com **ASP.NET Core**, aplicativos desktop modernos com **WPF** ou **Windows Forms** (e mais recentemente **MAUI** para multiplataforma), até serviços de backend robustos, jogos com a **Unity**, e soluções de inteligência artificial ou machine learning, o .NET oferece um ambiente robusto e versátil.

Evolução do .NET: Do Framework ao Multiplataforma

A história do .NET é marcada por uma evolução contínua para atender às demandas do mercado:

  • .NET Framework (Início dos Anos 2000): A versão original, dominante para o desenvolvimento de aplicações empresariais e desktop no ecossistema Windows. Era proprietário e limitado ao Windows.
  • .NET Core (2016): Uma reescrita revolucionária. Desenvolvido para ser **código aberto**, **modular**, e, o mais importante, **multiplataforma** (rodando em Windows, Linux e macOS). Focado em performance e cenários de nuvem.
  • .NET (a partir da versão .NET 5 em 2020): A unificação! A Microsoft consolidou o .NET Framework e o .NET Core sob um único nome, simplesmente **.NET**. Essa unificação representa o futuro da plataforma, combinando a vasta API do .NET Framework com a modularidade, o código aberto e a capacidade multiplataforma do .NET Core. É nesta versão moderna que focaremos nossos estudos.

**Por que essa evolução é importante?** A versão unificada do .NET oferece aos desenvolvedores a flexibilidade de construir aplicações que rodam em qualquer sistema operacional, aproveitando os benefícios de performance, modularidade e a vasta comunidade de código aberto.

1.2. O Que é C#?

**C# (pronuncia-se "C Sharp")** é a linguagem de programação primária e mais popular para desenvolvimento na plataforma .NET. Criada pela Microsoft, é uma linguagem:

  • **Moderna:** Possui recursos avançados e está em constante evolução, incorporando as melhores práticas de outras linguagens.
  • **Orientada a Objetos (POO):** Seu design é centrado em classes, objetos e princípios como herança, polimorfismo, encapsulamento e abstração, o que facilita a organização, a reutilização e a manutenção do código.
  • **Fortemente Tipada:** Toda variável deve ter um tipo de dado definido (ex: número inteiro, texto, booleano), o que ajuda a prevenir erros em tempo de compilação e torna o código mais robusto.
  • **De Propósito Geral:** Pode ser usada para desenvolver uma vasta gama de aplicações, como mencionado na seção anterior.

A sintaxe do C# é elegante e, para quem já teve contato com linguagens como Java ou C++, encontrará muitas semelhanças, o que facilita o aprendizado.

Aprender .NET e C# é um investimento estratégico para sua carreira no desenvolvimento de software pelas seguintes razões:

  • **Alta Demanda no Mercado:** Empresas de todos os portes, desde startups a grandes corporações, utilizam .NET para seus sistemas críticos. Há uma demanda consistente por profissionais qualificados em C#.
  • **Ecossistema Completo:** Além da linguagem e do framework, a Microsoft oferece um conjunto completo de ferramentas (como o **Visual Studio** e **Visual Studio Code**), serviços de nuvem (**Azure**) e uma vasta documentação de alta qualidade, facilitando o desenvolvimento e a implantação.
  • **Versatilidade:** Com .NET e C#, você pode atuar em diversas frentes: desenvolvimento web (front-end e back-end), desktop, mobile, jogos, microsserviços, IoT, inteligência artificial e até mesmo blockchain. Isso significa mais oportunidades de carreira e flexibilidade para escolher sua área de atuação.
  • **Performance e Escalabilidade:** O .NET moderno é conhecido por seu alto desempenho e capacidade de construir sistemas escaláveis, o que é crucial para aplicações que precisam lidar com um grande volume de usuários e dados.
  • **Comunidade Ativa e Suporte:** Possui uma comunidade global de desenvolvedores muito ativa, que contribui com recursos, resolve dúvidas e compartilha conhecimento. Além disso, conta com o suporte e a manutenção da Microsoft.
  • **Multiplataforma:** A capacidade de desenvolver e rodar aplicações em Windows, Linux e macOS abre portas para um mercado de trabalho mais amplo e projetos diversos.

1.5. Configurando o Ambiente de Desenvolvimento

Antes de mergulharmos no código, precisamos configurar seu ambiente. A ferramenta principal será o **Visual Studio Code** ou o **Visual Studio Community** (versão gratuita e completa).

Ferramentas Essenciais:

  1. **SDK do .NET:** O Software Development Kit (SDK) do .NET contém tudo o que você precisa para construir e executar aplicativos .NET, incluindo o runtime, bibliotecas e o compilador C#.
    • **Instalação:** Baixe a versão mais recente do .NET SDK no site oficial: dotnet.microsoft.com/download. Siga as instruções para seu sistema operacional (Windows, macOS, Linux).
    • **Verificação:** Após a instalação, abra o terminal (Prompt de Comando ou PowerShell no Windows, Terminal no Linux/macOS) e digite: `dotnet --version`. Se a versão for exibida, a instalação foi bem-sucedida.
  2. **Editor de Código (Recomendado):**
    • **Visual Studio Code (VS Code):** Leve, rápido e multiplataforma. É uma excelente escolha para iniciantes e profissionais.
      • **Instalação:** Baixe em code.visualstudio.com.
      • **Extensões Essenciais:** Após instalar o VS Code, procure e instale as seguintes extensões:
        • **C# (Microsoft):** Essencial para obter IntelliSense, depuração e outras funcionalidades de C#.
        • **C# Dev Kit (Microsoft):** Conjunto de extensões para desenvolvimento C# mais robusto.
        • **Debugger for C# (Microsoft):** Para depuração de código.
        • **NuGet Package Manager (Fuad Alabsi):** Para gerenciar pacotes.
    • **Visual Studio Community:** Uma IDE (Integrated Development Environment) completa e poderosa, exclusiva para Windows (com uma versão para Mac). É mais pesada que o VS Code, mas oferece uma experiência de desenvolvimento mais integrada para grandes projetos.
      • **Instalação:** Baixe em visualstudio.microsoft.com/vs/community/. Durante a instalação, certifique-se de selecionar as cargas de trabalho ("workloads") de **"Desenvolvimento de desktop com .NET"** e **"Desenvolvimento web e ASP.NET"**.

Criando seu Primeiro Projeto (Olá, Mundo!):

Vamos criar seu primeiro projeto de console para testar a instalação. Abra seu terminal (ou o terminal integrado no VS Code):


# Cria um novo diretório para seu projeto
mkdir MeuPrimeiroPrograma
cd MeuPrimeiroPrograma

# Cria um novo projeto de console em C#
dotnet new console

# Restaura as dependências (geralmente automático)
dotnet restore

# Executa o programa
dotnet run

Você deverá ver a saída "Hello, World!" no terminal. Isso significa que seu ambiente está configurado corretamente!

2. C# - Primeiros Passos: Sintaxe, Variáveis e Operadores

Agora que seu ambiente está configurado, vamos mergulhar na sintaxe básica do C# e nos blocos de construção fundamentais de qualquer programa: **variáveis** e **operadores**.

2.1. Estrutura de um Programa C#

Todo programa C# tem uma estrutura básica. Vamos analisar o "Olá, Mundo!" gerado automaticamente:


using System; // 1. Declaração de Namespace

namespace MeuPrimeiroPrograma // 2. Namespace do Projeto
{
    class Program // 3. Definição da Classe
    {
        static void Main(string[] args) // 4. Método Principal (Ponto de Entrada)
        {
            Console.WriteLine("Olá, Mundo!"); // 5. Instrução Executável
            Console.WriteLine("Bem-vindo ao C#!");
        }
    }
}
  1. `using System;`: A palavra-chave `using` é usada para **importar namespaces**. Namespaces são coleções de classes e outros tipos organizados para evitar conflitos de nomes e agrupar funcionalidades relacionadas. `System` é um namespace fundamental que contém classes para operações básicas como entrada/saída (como `Console`).
  2. `namespace MeuPrimeiroPrograma`: Define um **namespace** para o seu próprio código. Isso ajuda a organizar seus projetos, especialmente quando eles crescem em tamanho e complexidade ou quando você usa bibliotecas de terceiros.
  3. `class Program`: Todo o código em C# reside dentro de uma **classe**. Classes são modelos para criar objetos (veremos mais sobre isso em POO). A classe `Program` é uma convenção comum para abrigar o método de entrada do aplicativo.
  4. `static void Main(string[] args)`: Este é o **método principal** ou "ponto de entrada" do seu programa. Quando você executa um aplicativo C#, o Common Language Runtime (CLR) procura e executa este método primeiro.
    • `static`: Significa que você pode chamar este método sem criar uma instância da classe `Program`.
    • `void`: Indica que o método não retorna nenhum valor.
    • `Main`: O nome convencional para o método de entrada.
    • `(string[] args)`: Permite que você passe argumentos de linha de comando para o seu programa.
  5. `Console.WriteLine("Olá, Mundo!");`: Uma **instrução** que executa uma ação. `Console` é uma classe do namespace `System` que fornece métodos para entrada e saída de dados. `WriteLine` é um método que imprime uma linha de texto no console. Note o `;` (ponto e vírgula) ao final da instrução, que marca o seu término.

2.2. Variáveis e Tipos de Dados

**Variáveis** são "contêineres" nomeados que armazenam dados na memória do computador. Em C#, cada variável deve ter um **tipo de dado** definido antes de ser usada. Isso é o que chamamos de linguagem **fortemente tipada**.

Sintaxe de Declaração de Variáveis:


TipoDeDado nomeDaVariavel = valorInicial;

Categorias de Tipos de Dados em C#:

Os tipos de dados em C# são divididos em duas categorias principais, com diferenças importantes em como armazenam e manipulam dados na memória:

1. Tipos de Valor (Value Types)

Armazenam diretamente seus valores na memória onde são declarados (geralmente na pilha - "stack"). Quando você atribui um tipo de valor a outra variável, uma **cópia do valor** é feita.

  • **Inteiros:** Armazenam números inteiros (positivos, negativos e zero).
    • `byte` (0 a 255)
    • `sbyte` (-128 a 127)
    • `short` (-32.768 a 32.767)
    • `ushort` (0 a 65.535)
    • `int` (o mais comum, aproximadamente ±2 bilhões)
    • `uint` (0 a 4 bilhões)
    • `long` (para números muito grandes, aproximadamente ±9 quintilhões)
    • `ulong` (0 a 18 quintilhões)
  • **Ponto Flutuante:** Armazenam números com casas decimais.
    • `float` (precisão de 7 dígitos, sufixo `f` ou `F` é obrigatório: `3.14f`)
    • `double` (o mais comum, precisão de 15-16 dígitos)
    • `decimal` (alta precisão para cálculos financeiros, sufixo `m` ou `M` é obrigatório: `123.45m`)
  • **Caracteres:**
    • `char` (um único caractere Unicode, entre aspas simples: `'A'`, `'7'`, `'@'`)
  • **Booleano:**
    • `bool` (apenas dois valores: `true` ou `false`)

int idade = 30; // Tipo inteiro
double peso = 75.5; // Tipo ponto flutuante de dupla precisão
char primeiraLetra = 'C'; // Tipo caractere
bool estudanteAtivo = true; // Tipo booleano
decimal precoProduto = 199.99m; // Tipo decimal para precisão financeira

2. Tipos de Referência (Reference Types)

Não armazenam o valor diretamente, mas sim uma **referência (endereço de memória)** para o local onde o valor real está armazenado (geralmente no "heap"). Quando você atribui um tipo de referência a outra variável, a referência é copiada, não o valor em si. Isso significa que ambas as variáveis apontarão para o **mesmo local de memória**.

  • `string`: Uma sequência de caracteres Unicode (texto), entre aspas duplas: `"Olá, C#!"`, `"DIO Bootcamp"`.
  • `object`: O tipo base para todos os outros tipos em C#. Qualquer tipo pode ser implicitamente convertido para `object`.
  • Classes, Interfaces, Delegates, Arrays (sim, arrays são tipos de referência, mesmo que contenham tipos de valor!).

string nomeCompleto = "Ana Paula Santos"; // Tipo string
object valorGenerico = 123; // Tipo object pode armazenar qualquer coisa
int[] meusNumeros = { 1, 2, 3 }; // Array de inteiros (tipo de referência)

A compreensão da diferença entre tipos de valor e tipos de referência é crucial para evitar comportamentos inesperados no seu código, especialmente ao passar variáveis para métodos.

2.3. Operadores em C#

Operadores são símbolos que dizem ao compilador para realizar manipulações específicas em valores ou variáveis.

a) Operadores Aritméticos

Usados para realizar operações matemáticas básicas.

Operador Nome Exemplo Resultado
`+` Adição `int resultado = 5 + 3;` `8`
`-` Subtração `int resultado = 10 - 4;` `6`
`*` Multiplicação `int resultado = 2 * 6;` `12`
`/` Divisão `double resultado = 10.0 / 3;` `3.333...`
`%` Módulo (resto da divisão) `int resultado = 10 % 3;` `1`
`++` Incremento (adiciona 1) `int x = 5; x++;` `x` agora é `6`
`--` Decremento (subtrai 1) `int y = 8; y--;` `y` agora é `7`

**Observação sobre divisão:** A divisão de dois inteiros em C# sempre resulta em um inteiro (a parte fracionária é truncada). Para obter um resultado com casas decimais, um dos operandos deve ser de ponto flutuante (`double`, `float` ou `decimal`) ou você deve fazer um **casting** explícito. 


int num1 = 10;
int num2 = 3;

int divisaoInteira = num1 / num2; // Resultado: 3
double divisaoFlutuante = (double)num1 / num2; // Resultado: 3.3333333333333335

b) Operadores de Atribuição

Usados para atribuir valores a variáveis. O mais básico é `=`, mas existem atalhos.

Operador Exemplo Equivalente a
`=` `int x = 10;` Atribuição simples
`+=` `x += 5;` `x = x + 5;`
`-=` `x -= 2;` `x = x - 2;`
`*=` `x *= 3;` `x = x * 3;`
`/=` `x /= 2;` `x = x / 2;`
`%=` `x %= 4;` `x = x % 4;`

c) Operadores de Comparação (Relacionais)

Usados para comparar dois valores e retornar um resultado booleano (`true` ou `false`).

Operador Nome Exemplo Resultado (se a=5, b=3)
`==` Igual a `a == b` `false`
`!=` Diferente de `a != b` `true`
`>` Maior que `a > b` `true`
`<` Menor que `a < b` `false`
`>=` Maior ou igual a `a >= b` `true`
`<=` Menor ou igual a `a <= b` `false`

Usados para combinar ou modificar expressões booleanas.
